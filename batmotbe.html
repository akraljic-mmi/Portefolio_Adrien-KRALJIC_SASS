Illutration batiment Montbéliard
Présentation :

Le but de cet exercice est de concevoir une illustration Vectorielle grâce à Illustrator. La suite des consignes est de choisir trois bâtiments de Belfort et de Montbéliard pour les faire en flat-design. Nous devions ensuite les animer.

Procédé de création :

Dans un premier temps il faut concevoir l'illustration vectoriel grace a illustrator. Dans un deuxieme temps il nécessaire d'exporter l'illustration en format SVG pour ensuite l'intégrer

Compétences mises en œuvre :

Illustration vectorielle via Illustrator
Intégration d'illustration SVG dans un format web
Animation SVG par le biet des languages CSS, Javascript.

Voir l'annimation

* Attention cette illustration n'est pas responsive
